Quick Start

Add these two things to your HTML:

<!-- 1. Add the container with your config -->
<div id="kaspa-pay" 
     data-address="kaspa:your-address-here"
     data-amount="10"
     data-item="Coffee"
     data-theme="dark">
</div>

<!-- 2. Include the script -->
<script src="kaspa-pay-modal.js"></script>

Configuration Options

data-address  // Required - Your kaspa: address
data-amount   // Amount in KAS
data-item     // Item/product name
data-theme    // "light" or "dark"

JavaScript API

For dynamic payments, use the JavaScript API:

KaspaPay.open({
    address: 'kaspa:qr...',
    amount: 5,
    item: 'Donation',
    theme: 'dark',
    onSuccess: function(result) {
        console.log('Paid!', result.txid);
    }
});

Use Cases

Donation buttons - Add a tip jar to your blog or content

Simple e-commerce - Sell products without a payment processor

Per-user payments - Each page can have a different receiving address
